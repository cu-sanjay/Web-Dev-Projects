# Escape Room Designer

Design and test custom escape rooms. Choose from 5 puzzle types (Riddle, Number Lock, Hidden Key, Pattern Match, Cipher), assign them to 4 slots, and test solutions.

## Features
- 5 puzzle types with descriptions
- 4 puzzle slots in a 2×2 grid
- Room name, difficulty, and time limit settings
- Solution testing and export functionality

## Local Deployment
```
cd "Escape Room Designer"
start index.html
```
