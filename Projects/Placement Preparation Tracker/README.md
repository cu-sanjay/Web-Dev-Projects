# 🚀 Placement Preparation Tracker

A responsive and interactive Placement Preparation Tracker using HTML, CSS, and JavaScript that helps students and job seekers organize, monitor, and improve their placement preparation journey.

## Features

- **Preparation Dashboard**: Overall placement readiness score with a progress bar tracking all tasks.
- **DSA Tracker**: Track coding problems and Data Structures & Algorithms preparation.
- **Aptitude Preparation**: Track Quantitative, Logical, and Verbal reasoning practice.
- **Core Subjects**: Manage preparation for OS, DBMS, Networks, etc.
- **Resume Preparation**: Checklist for resume and portfolio readiness.
- **Mock Interviews**: Log mock interviews, feedback, and communication skills practice.
- **Company Specific**: Notes, experiences, and specific preparation for target companies.
- **Dark/Light Mode**: User preference for theme, saved locally.
- **Data Persistence**: Uses LocalStorage to automatically save all your preparation data.

## How to Run

1. Clone this repository.
2. Navigate to `Projects/placement-preparation-tracker`.
3. Open `index.html` in your web browser.
4. Add your daily preparation targets under each category and check them off to watch your overall readiness grow!

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla, DOM manipulation, LocalStorage)
