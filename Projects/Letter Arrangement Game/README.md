# Word Scramble Game

A modern word puzzle game where players rearrange scrambled letters to form valid words.

## Features

- Modern Glassmorphism Design
- Difficulty Levels
- Score Tracking
- Countdown Timer
- Hint System
- Skip Word Feature
- Local Storage High Score
- Responsive Layout
- Dark/Light Mode Toggle
- Animated Feedback

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Play

1. Read the scrambled word.
2. Type the correct word.
3. Click Check Answer.
4. Earn points for correct answers.
5. Use hints if needed.
6. Beat your high score.

## Installation

Simply open `index.html` in any browser.
