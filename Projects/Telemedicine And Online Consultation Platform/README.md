# Telemedicine & Online Consultation Platform

A comprehensive front-end web application that streamlines the process of finding doctors, scheduling virtual consultations, and managing personal healthcare records. 

## Features
- **Patient Dashboard**: Track upcoming virtual appointments, active prescriptions, and recent lab results in one place.
- **Find a Doctor**: Browse and filter available healthcare specialists by department, rating, and availability.
- **Appointment Scheduler**: A clear interface for booking online consultations.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
