# Learning Management System (LMS)

A comprehensive front-end web application aimed at facilitating online education. The platform includes a dashboard for students to view their enrolled courses, track their progress, access course materials, and manage their assignments.

## Features
- **Student Dashboard**: Overview of current courses, recent grades, and upcoming deadlines.
- **Course Catalog**: Browse available courses to enroll in.
- **Course View**: Detailed view of a course, including modules, lectures, and assignments.
- **Progress Tracking**: Visual indicators of completion status for modules and overall courses.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
