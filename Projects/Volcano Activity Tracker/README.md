# Volcano Activity Tracker

Track global volcanic activity with detailed profiles, eruption histories, and risk assessments.

## Features
- 8 volcanoes across different regions and types
- Filter by risk level (Low, Moderate, High, Critical) and region
- Detailed profiles with elevation, type, status, and descriptions
- Eruption history with VEI ratings and descriptions
- Color-coded risk indicators

## Local Deployment
```
git clone <repository-url>
cd "Volcano Activity Tracker"
start index.html
```
