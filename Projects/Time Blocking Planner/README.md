# 📅 Time Blocking Planner

Plan daily schedules using time blocks, priorities, and calendar-based productivity techniques.

## Features
- ➕ Add tasks with start and end times
- 🏷️ Assign priority (High, Medium, Low)
- 📅 Visual schedule layout
- 💾 LocalStorage persistence
- 🎨 Attractive, responsive UI

## Usage
Open `index.html` in your browser.  
Enter task details, add blocks, and view your daily schedule.
