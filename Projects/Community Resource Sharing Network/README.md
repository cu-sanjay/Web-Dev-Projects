# Community Resource Sharing Network

A cutting-edge, ultra-modern web platform designed to facilitate the sharing of physical resources (tools, equipment, books) within a local community. It features a stunning glassmorphism UI to provide an exceptional user experience.

## Features
- **Ultra-Modern UI/UX**: Premium aesthetic featuring glassmorphism, dynamic gradients, smooth micro-animations, and a responsive layout.
- **Resource Feed**: Browse items available for borrowing within your community.
- **Lend & Borrow**: Easily request to borrow items, or list your own items to lend.
- **My Inventory**: A dedicated space to manage the resources you've lent out and those you are currently borrowing.
- **Trust System**: View the reputation and reliability of community members.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, Glassmorphism, CSS Variables, Animations)
- Vanilla JavaScript
