Regex Cheat Sheet

Character Classes

  • . Any character except newline
  • \w Word character [a-zA-Z0-9_]
  • \d Digit [0-9]
  • \s Whitespace character
  • [abc] Any of a, b, or c
  • [^abc] Not a, b, or c
  • [a-z] Character between a & z

Quantifiers

  • * 0 or more
  • + 1 or more
  • ? 0 or 1
  • {3} Exactly 3
  • {3,} 3 or more
  • {3,5} Between 3 and 5

Anchors

  • ^ Start of string/line
  • $ End of string/line
  • \b Word boundary
