# 🔍 Regex Tester & Visualizer

Regular Expressions are powerful but notoriously difficult to master. This browser-based developer utility helps you understand, test, and visualize exactly what your Regex matches in real-time. 

## ✨ Features

- **Live Highlighting:** Type your test string and see exactly where the Regex matches instantly.
- **Syntax Error Handling:** Typing invalid regex won't crash the app; it gracefully handles errors in the background until the syntax is valid.
- **Regex Flags:** Easily toggle global (`g`), case-insensitive (`i`), and multiline (`m`) flags.
- **Cheat Sheet Sidebar:** A quick reference guide to common Regex tokens built right into the UI.
- **Custom Overlay System:** Utilizes a custom-built textarea overlay to seamlessly highlight text as you type.

## 🛠️ Built With

- **HTML5:** Semantic layout with a split-pane design.
- **CSS3:** A sleek, developer-focused dark theme with precise absolute positioning for the highlight overlay.
- **Vanilla JavaScript:** Event listeners, Regex compilation, HTML entity escaping, and dynamic DOM manipulation.

## 🚀 How to Run

1. Navigate to the `Projects/Regex Tester` directory.
2. Open `index.html` in your web browser.
3. Type a regular expression in the top input and type text below to see live highlighting!
