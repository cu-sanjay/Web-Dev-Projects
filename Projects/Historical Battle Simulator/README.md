# Historical Battle Simulator

Simulate historical battles by configuring two armies with infantry, cavalry, and archers. Choose terrain (Plains, Forest, Hills, River Crossing, Siege) to affect unit performance and see battle outcomes.

## Features
- Two configurable armies with 3 unit types each
- 5 terrain types with unit-specific bonuses
- Visual army representation on battlefield
- Casualty tracking and victory determination

## Local Deployment
```
cd "Historical Battle Simulator"
start index.html
```
