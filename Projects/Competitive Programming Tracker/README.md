# Competitive Programming Tracker

A competitive programming progress dashboard inspired by Codeforces and LeetCode, featuring topic-wise progress tracking, streak monitoring, and problem management — all running entirely client-side.

---

## Features

- **Dashboard Overview** — Total solved, contest rating, global rank, difficulty breakdown (Easy/Medium/Hard)
- **Monthly Activity Chart** — Bar chart showing problems solved per month
- **Recent Submissions** — Last 5 submissions with verdict (Accepted/WA) and topic tags
- **Topic-Wise Progress** — 10 tracked topics with solved/total counts and percentage bars
- **Streak Tracker** — Current streak, longest streak, total active days, consistency percentage, and 30-day activity heatmap
- **Problem Library** — 30 sample problems filterable by difficulty, with status indicators

## Streak System

- **Current Streak** — Consecutive days with at least one solved problem
- **Longest Streak** — All-time best streak (47 days)
- **Consistency** — Percentage of active days in the tracked period
- **Activity Heatmap** — Color-coded 30-day grid (darker = more problems solved)

## Local Deployment

```bash

start index.html
```

Open `index.html` directly in any browser.
