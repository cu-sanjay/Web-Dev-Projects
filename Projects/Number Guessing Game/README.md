# Number Guessing Game

A cyberpunk number-guessing game built with HTML5, CSS3, and vanilla JavaScript.

## Features

- Random secret number between 1 and 100.
- Dynamic range contraction: bounds update after every guess.
- Color-coded feedback: amber for high, cyan for low, emerald for correct.
- Scrollable guess history log with relational status labels.
- Attempt counter and all-time best record via `localStorage`.
- Zero-refresh reset and victory modal overlay.

## Run it

Open `index.html` in any modern browser.
