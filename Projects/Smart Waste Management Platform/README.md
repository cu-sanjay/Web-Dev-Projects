# Smart Waste Management Platform

An eco-friendly digital platform designed to optimize urban waste collection by connecting citizens directly with municipal sanitation departments.

## Features
- **Citizen Dashboard**: A clean interface for residents to check their neighborhood's waste collection schedule, report illegal dumping, and schedule special bulk pickups.
- **Sanitation Dept Portal**: A secure dashboard for city workers to monitor smart bin fill levels, track active collection routes, and manage citizen reports.
- **Interactive Role Switcher**: Seamlessly toggle between the Citizen View and the Sanitation Dept View to experience both sides of the platform.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
