# 🎯 Goal Achievement Tracker

Set goals, track milestones, monitor progress, and visualize achievements over time.

## Features
- ➕ Add goals with milestones
- ✅ Mark milestones as completed
- 📊 Visual progress bars
- 📈 Chart visualization of progress
- 💾 LocalStorage persistence
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Add goals, complete milestones, and track your progress visually.
