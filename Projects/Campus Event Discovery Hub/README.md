# Campus Event Discovery Hub

A centralized, interactive web application designed to help college students discover, filter, and track upcoming campus events, club meetings, and academic workshops. Built with purely Vanilla JavaScript, HTML, and CSS.

## Features
- **Dynamic Event Feed**: Browse a grid of upcoming events hosted on campus.
- **Advanced Filtering**: Filter events by category (Social, Academic, Sports, Arts) to find exactly what interests you.
- **Search Functionality**: Instantly search for specific events or club names.
- **Event Booking/RSVP**: Click to "RSVP" to an event and see the attendee count update in real-time.
- **Responsive Layout**: Clean, accessible design that adapts perfectly to both desktop and mobile screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
