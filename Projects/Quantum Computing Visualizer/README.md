# Quantum Computing Visualizer

Interactive visualization of a single qubit's quantum state. Apply Hadamard, Pauli-X/Y/Z, S, and T gates and observe changes on the Bloch sphere, probability bars, and the quantum circuit diagram.

## Features
- Bloch sphere representation of qubit state
- 6 quantum gates (H, X, Y, Z, S, T)
- Real-time probability amplitude display
- Visual quantum circuit with gate annotations

## Local Deployment
```
cd "Quantum Computing Visualizer"
start index.html
```
