# ⏰ Digital Clock with Themes

A live digital clock displaying hours, minutes, seconds, and date with switchable light/dark/neon themes — built using HTML, CSS, and JavaScript.

## Features
- 🕒 Real-time digital clock
- 📅 Date display
- 🌞 Light theme
- 🌙 Dark theme
- 💡 Neon theme with glowing effects
- 💾 Theme persistence via LocalStorage
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Switch between themes using the buttons and enjoy the live clock.
