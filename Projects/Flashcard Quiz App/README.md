# 📚 Flashcard Quiz App

An interactive flashcard quiz app where users can add custom Q&A cards, flip them to reveal answers, and track their score.

## Features
- ➕ Add custom flashcards
- 🔄 Flip animation to reveal answers
- 🎲 Shuffle mode (optional extension)
- 🏆 Score tracking
- 💾 LocalStorage persistence

## Usage
Open `index.html` in your browser.  
Add questions and answers, click cards to flip, and track your score.
