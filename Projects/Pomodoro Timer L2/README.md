# ⏳ Pomodoro Timer

A productivity timer with **25/5 minute cycles**, session counter, and audio notification.  
Built using **HTML, CSS, and JavaScript**.

## Features
- 25‑minute work sessions
- 5‑minute breaks
- Start / Pause / Reset controls
- Audio notification when session ends
- Session counter to track progress
- Responsive, attractive UI

## Usage
Open `index.html` in your browser and start the timer.
