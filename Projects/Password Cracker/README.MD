# 🔐 Password Cracker


## Description

Added Password Cracker, an interactive puzzle game where users receive clues about a secret password and must correctly guess it. The game challenges logical thinking, deduction, and word analysis skills through progressively difficult hints.

## Features

* Random password generation
* Hint-based gameplay
* Password guessing system
* Progressive difficulty levels
* Score tracking
* Instant answer validation
* Multiple rounds of challenges

## Example

**Hints:**

* Length = 5
* Contains a vowel
* Starts with B

**Answer:** Bread

## Benefits

* Enhances logical reasoning
* Improves deduction and problem-solving skills
* Encourages critical thinking
* Provides an engaging puzzle-solving experience
