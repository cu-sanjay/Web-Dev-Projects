# Student Health Assistant

A comprehensive front-end web application aimed at helping college students manage their physical and mental health. The app includes features for booking medical appointments, tracking daily wellness routines (sleep, water intake, exercise), and accessing emergency contacts quickly.

## Features
- **Appointment Booking**: Schedule appointments with campus clinic doctors.
- **Wellness Tracker**: Log and monitor daily water intake, sleep hours, and exercise.
- **Health Tips**: Receive daily randomized health and wellness tips.
- **Emergency Contacts**: Quick access to campus security, medical emergencies, and mental health hotlines.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
