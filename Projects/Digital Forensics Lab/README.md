# Digital Forensics Lab

Interactive forensic analysis workspace with four tools: Hex Viewer, Timeline Reconstruction, Metadata Extraction, and File Recovery. Analyze evidence files (image, log, memory dump, USB drive).

## Features
- Hex viewer with offset, byte, and ASCII display
- Timeline event reconstruction
- Metadata grid for each evidence file
- File recovery status with recoverable files list
- 4 evidence file types

## Local Deployment
```
cd "Digital Forensics Lab"
start index.html
```
