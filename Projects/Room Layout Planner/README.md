# 🛋️ Room Layout Planner

A responsive Room Layout Planner that enables users to design and visualize room layouts using an intuitive drag-and-drop interface. Try different room arrangements without physically moving heavy furniture!

## Features

- **Interactive Canvas**: Set the width and height of your room.
- **Furniture Library**: Drag and drop items like beds, sofas, tables, chairs, wardrobes, and plants into the room.
- **Drag & Drop**: Easily move placed items around on a snap-to-grid layout.
- **Rotation**: Double-click any placed item to rotate it.
- **Deletion**: Select an item and press `Delete` or `Backspace` to remove it from the room.
- **Space Analysis**: See live statistics for total room area and the number of items placed.
- **Theme Support**: Includes a Dark Mode that saves locally.
- **Data Persistence**: Uses LocalStorage so your perfect layout isn't lost if you refresh the page.

## How to Run

1. Clone the repository.
2. Navigate to `Projects/room-layout-planner/`.
3. Open `index.html` in your web browser.

## Technologies Used

- HTML5 (Drag and Drop API)
- CSS3 (Variables, Grid, Flexbox)
- Vanilla JavaScript (LocalStorage API, DOM manipulation, Events)
