# Campus Resource Exchange Platform

A centralized, interactive platform designed for college students to easily list, discover, and trade textbooks, electronics, and other academic supplies right on their campus.

## Features
- **Item Listings**: Browse through available textbooks, electronics, and study materials.
- **Search & Filter**: Quickly find exactly what you need by searching titles or filtering by category.
- **Post a Resource**: Easily add new items to the marketplace with titles, categories, condition, and value.
- **Interactive Exchange UI**: Clean, responsive layout showcasing active listings as cards.

## Run It
Simply double-click or open `index.html` in any modern web browser to start using the platform. No server or package manager required.

## What I Learned
- Building a dynamic, client-side data store using Vanilla JavaScript.
- Implementing real-time DOM filtering and search functionality.
- Creating a clean, responsive card-based layout using CSS Grid and Flexbox.
- Managing interactive modals for form submissions.
