# 🔌 Digital Circuits Lab & Logic Gate Simulator

An interactive educational playground for building and simulating logic gates and digital circuits. Spawn gates, wire inputs and outputs, track Boolean truth tables, and complete logical lab challenges.

---

## 🌟 Key Features

1. **Interactive Node Editor Canvas**:
   - Spawn components (AND, OR, NOT, NAND, NOR, XOR, Input Switch, Output Bulb).
   - Draw wire connections dynamically between terminals.

2. **Real-time Simulation & Active Wire Tracing**:
   - Live color coding: High/True wires glow bright cyan, Low/False wires remain dark gray.

3. **Auto-generated Truth Tables**:
   - Select any gate to see its truth table updated dynamically based on real-time inputs.

4. **CS Educational Challenges (Labs)**:
   - Interactive tasks (e.g., "Build an OR Gate using NAND gates", "Create a Half-Adder") with automated verification.

---

## 🚀 How to Run

1. Simply open [index.html](file:///index.html) in any modern web browser.
2. Select components from the sidebar and place them on the canvas.
