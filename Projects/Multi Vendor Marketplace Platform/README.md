# Multi Vendor Marketplace Platform

A robust, front-end web application featuring a unified portal with dedicated, role-based dashboards for Admins, Vendors, and Customers.

## Features
- **Admin Dashboard**: Oversee the entire marketplace. Monitor total sales, approve new vendors, and manage global platform settings.
- **Vendor Dashboard**: Dedicated space for sellers to manage their product catalog, view store analytics, and process pending orders.
- **Customer Dashboard**: A clean interface for shoppers to track their recent orders, manage their wishlist, and view purchase history.
- **Role Switching**: Seamlessly toggle between Admin, Vendor, and Customer views to experience the entire ecosystem.
- **Responsive UI**: Built with modern CSS to work beautifully on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
