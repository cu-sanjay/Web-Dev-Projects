# Protein Structure Viewer

Interactive 3D protein structure visualizer with four proteins (Hemoglobin, Insulin, Collagen, Lysozyme), three display modes (ribbon, spheres, wireframe), and drag-to-rotate controls.

## Features
- 4 preloaded protein structures
- Ribbon, sphere, and wireframe display modes
- Real-time 3D rotation and zoom
- Detailed per-protein info panel

## Local Deployment
```
cd "Protein Structure Viewer"
start index.html
```
