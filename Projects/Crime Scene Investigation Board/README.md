# Crime Scene Investigation Board

Interactive investigation board. Drag evidence items and suspects onto the board, auto-connect related evidence, add timeline events, and visualize connections.

## Features
- 6 evidence types (fingerprint, footprint, DNA, weapon, letter, photo)
- 3 suspect profiles
- Drag-and-drop placement on investigation board
- Auto-connection lines between related evidence
- Custom timeline event log

## Local Deployment
```
cd "Crime Scene Investigation Board"
start index.html
```
