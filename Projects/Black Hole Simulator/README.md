# Black Hole Simulator

An interactive canvas-based visualization of a black hole's event horizon, accretion disk, and gravitational lensing effect. Adjust mass, disk density, and rotation speed in real time.

## Features
- Real-time canvas rendering with gravitational lensing
- Adjustable mass, accretion disk, and spin parameters
- Photon ring and event horizon visualization
- Background star distortion near the black hole

## Local Deployment
```
cd "Black Hole Simulator"
start index.html
```
