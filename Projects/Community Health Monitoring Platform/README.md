# Community Health Monitoring Platform

A comprehensive front-end web application that visualizes public health data, monitors outbreaks, and tracks medical resource distribution across different districts.

## Features
- **Health Dashboard**: Get a high-level overview of active cases, recovery rates, and critical alerts.
- **Outbreak Map**: Monitor disease spread across various zones with visual indicators.
- **Resource Tracker**: Keep tabs on hospital bed availability, vaccine stock, and medical personnel distribution.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
