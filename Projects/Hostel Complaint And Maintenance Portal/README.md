# Hostel Complaint & Maintenance Portal

A utility application designed for college hostel residents to easily log, track, and manage their maintenance complaints (e.g., plumbing, electrical, carpentry).

## Features
- **Complaint Logging Form**: Residents can submit complaints specifying their room number, issue category, and a description.
- **Dynamic Status Tracking**: View all logged complaints in a dashboard with visual status indicators (Pending, In Progress, Resolved).
- **Status Toggling**: Users can interact with the dashboard to update the status of their complaints, simulating administrative action.
- **Responsive Interface**: Clean, modern design accessible on both desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
