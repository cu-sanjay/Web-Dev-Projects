# BiMemory Game

A dark-themed browser game that tests your short-term memory.

## How to Play

1. Click **Start Game**.
2. A random number appears for 2 seconds.
3. The number disappears and you must type the exact number from memory.
4. If your answer is correct, you move to the next level and see confetti.
5. If your answer is wrong, you lose one life.
6. The game ends when you lose all lives.

## Features

- Clean dark UI with modern glass-style cards
- Increasing number length as levels advance
- 3 lives per run
- Confetti animation on success
- Best level track record

## Files

- `index.html` — game layout and structure
- `style.css` — dark theme styles and responsive layout
- `script.js` — game logic and behavior
- `project.json` — project metadata for the OpenStudio collection

## Notes

This game is built with vanilla JavaScript, HTML, and CSS.
