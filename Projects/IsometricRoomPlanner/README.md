# 📐 Isometric Room Planner & 2.5D Designer

An interactive client-side 2.5D layout sandbox designed to plan interior layouts. Design customized grid blueprints, place and rotate furniture components, track footprint metrics, and export floor plans.

---

## 🌟 Key Features

1. **Interactive 2.5D Isometric Grid**:
   - Render mathematical room layouts with dynamic walls.
   - Snapping mechanics for precise alignment.

2. **Drag-and-Drop furniture library**:
   - Includes standard interior shapes (bed, table, chair, couch, desk, plant, TV console).
   - Dynamic 90-degree rotation.

3. **Room Dimensions controls**:
   - Dynamically adjust floor width, length, and floor color themes.

4. **Footprint Calculator**:
   - Real-time statistics: total floor space, occupied footprint, and remaining clearance.

5. **Design Exporter**:
   - Export your layout as an image directly.

---

## 🚀 How to Run

1. Simply open [index.html](file:///index.html) in any modern web browser.
2. Design and customize elements directly on the canvas.
