# Optics Playground

An interactive ray optics simulator where you drag a torch (light source) and choose between mirrors (concave/convex) or lenses (converging/diverging). Watch real‑time reflection/refraction behavior and get live physics calculations.

## Features

- **Draggable object** – click and drag the torch to change object distance.
- **Switch between mirror & lens** – concave/convex mirrors, converging/diverging lenses.
- **Real‑time ray drawing** – simplified principal rays that update instantly.
- **Live calculations** – object distance, image distance, magnification, image nature, focal length.
- **Interactive canvas** – no gradients, smooth frame updates.
- **Beginner‑friendly physics** – uses mirror/lens equation (1/f = 1/u + 1/v).

## Tech Used

- **HTML5** – semantic layout, canvas element.
- **CSS3** – flexbox, clean card design, no gradients.
- **JavaScript (ES6)** – canvas rendering, drag‑and‑drop, real‑time formula evaluation, event handling.
