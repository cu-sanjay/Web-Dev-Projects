# Detective Investigation Simulator

Interactive detective game featuring crime scene investigation with clickable objects, clue discovery, suspect tracking, and accusation mechanics.

## Features
- 2 cases (Stolen Diamond, Missing Will)
- 4 interactive crime scene objects
- 3 discoverable clues per case
- 3 suspects with clue linkage tracking
- Accusation system with correct culprit

## Local Deployment
```
cd "Detective Investigation Simulator"
start index.html
```
