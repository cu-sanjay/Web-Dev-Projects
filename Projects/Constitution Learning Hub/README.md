# Constitution Learning Hub

Explore the United States Constitution through three interactive modes: browse Articles, review Amendments, and test your knowledge with a quiz.

## Features
- **Articles** — Read all 7 articles with detailed explanations and search/filter by part
- **Amendments** — Browse key amendments from the Bill of Rights to the 26th Amendment
- **Quiz Mode** — Random 10-question quizzes testing constitutional knowledge

Open `index.html` in any browser to start learning.
