# Micro-Investment Simulator

A modern, interactive web application that allows users to simulate the growth of small, recurring investments over time using compound interest, portfolio allocation, and dynamic forecasting.

## Features
- **Investment Dashboard**: View your simulated portfolio balance, projected returns, and overall growth timeline.
- **Portfolio Allocation**: Distribute your funds across different asset classes (Stocks, Bonds, Real Estate, Crypto) with an interactive UI.
- **Round-Up Simulator**: Simulate "spare change" round-ups from daily purchases and see how they contribute to your long-term wealth.
- **Compound Interest Calculator**: Adjust monthly contributions, interest rates, and time horizons to forecast future values.
- **Responsive UI**: Built with a sleek dark/light theme supporting both desktop and mobile layouts.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
