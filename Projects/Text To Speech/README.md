# Text To Speech

A simple text to speech app that reads out the users line of input.

## Features

- Text box which gets input from the user.
- A button which prompts the app to read out the users entry.
- Simple interface with no clutter.

## Run it

Open `index.html` in any modern browser.

## What it shows

- A card containing a text box, waiting for user input.
- A responsive button which prompts JavaScript action.
