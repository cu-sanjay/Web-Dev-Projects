# World Heritage Explorer

Discover UNESCO World Heritage Sites with an interactive browser-based explorer. Search by name or country, filter by region and type, and browse detailed information about each site.

## Features
- 24 UNESCO World Heritage Sites across all continents
- Filter by region (Asia, Europe, Africa, Americas, Oceania)
- Filter by type (Cultural, Natural, Mixed)
- Search by site name or country
- Site cards with icons, descriptions, and UNESCO inscription year

## Local Deployment
```
git clone <repository-url>
cd "World Heritage Explorer"
start index.html
```
