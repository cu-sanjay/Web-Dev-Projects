# System Design Interview Playground

Interactive architecture diagram builder for system design practice. Add 7 component types (Client, Server, Database, Cache, Queue, CDN, Load Balancer) and use 3 preset architectures.

## Features
- 7 draggable system components
- 3 preset architectures (Chat App, E-Commerce, Video Platform)
- Automatic edge connections
- Click-to-remove and export functionality

## Local Deployment
```
cd "System Design Interview Playground"
start index.html
```
