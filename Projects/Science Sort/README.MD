#  Science Sort

## Description

Added Science Sort, an educational classification game where users sort scientific facts into the correct branch of science: Physics, Chemistry, or Biology. The game helps reinforce scientific knowledge through interactive learning.

## Features

* Fact classification challenges
* Physics, Chemistry, and Biology categories
* Drag-and-drop or selection-based sorting
* Score tracking system
* Instant answer validation
* Progressive difficulty levels
* Interactive learning experience

## Example

* "Photosynthesis occurs in plants" → **Biology**
* "Force equals mass × acceleration" → **Physics**
* "Atoms combine to form molecules" → **Chemistry**

## Benefits

* Strengthens scientific understanding
* Improves classification and reasoning skills
* Reinforces subject knowledge
* Makes science learning engaging and interactive
