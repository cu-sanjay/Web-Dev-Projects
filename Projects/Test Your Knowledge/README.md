# 🧠 Test Your Knowledge

## Description

Added a user-friendly General Knowledge application that allows users to test and improve their knowledge through engaging quizzes and interactive questions across various topics.

## Features

* General knowledge quizzes
* Multiple-choice questions
* Score tracking system
* Instant answer feedback
* User-friendly interface
* Interactive learning experience

## Benefits

* Expands general knowledge
* Encourages continuous learning
* Provides an engaging quiz experience
* Helps users assess their knowledge level

## Future Improvements

* Category-based quizzes
* Difficulty levels
* Leaderboard system
* Daily challenges
* Achievement badges
* Personalized question recommendations

## Use Cases

* Knowledge assessment
* Educational practice
* Competitive exam preparation
* Daily learning activities
