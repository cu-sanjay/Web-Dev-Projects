# Skill Development & Career Tracking Platform

A comprehensive front-end web application aimed at helping individuals track their career progression and develop new skills. The platform provides tools for visualizing learning paths, managing skill milestones, and monitoring career goals.

## Features
- **Interactive Dashboard**: An overview of current career goals, skill progress, and upcoming milestones.
- **Learning Paths**: Structured roadmaps mapping out the skills needed for specific career trajectories.
- **Skill Tracker**: Visual progression indicators for individual skills (e.g. beginner to expert).
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
