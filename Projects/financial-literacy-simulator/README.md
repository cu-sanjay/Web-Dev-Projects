# Financial Literacy Simulator

An interactive Financial Literacy Simulator that allows users to manage a virtual income, make spending decisions, save money, invest funds, handle emergencies, and work toward long-term financial goals.

## Overview
Built with **React** and **Vite**, this simulator gamifies the personal finance journey. It simulates real-life financial situations and helps users understand how daily financial decisions affect their future wealth, financial stability, and overall financial health.

## Features
- **Virtual Profile**: Set up your character's starting income and financial goals.
- **Budgeting**: Allocate your monthly income across housing, food, transportation, and more.
- **Investments**: Test your strategies with virtual Stocks, Real Estate, and Fixed Deposits.
- **Life Scenarios**: Handle unexpected emergencies like job loss or medical bills.
- **Debt Management**: Practice paying down student loans or credit card debt.
- **Analytics Dashboard**: Track your net worth, savings rate, and overall financial health score over time.

## Running the project locally
1. Ensure you have Node.js installed.
2. Navigate to this directory in your terminal: `cd Projects/financial-literacy-simulator`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
