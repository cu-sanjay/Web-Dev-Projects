# Lost Civilization Explorer

Explore 4 civilizations: Mayan, Indus Valley, Minoan, and the legendary Atlantis. Each entry features facts, key achievements, period details, and unanswered mysteries.

## Features
- 4 civilizations with detailed overviews
- Visual art representation in SVG
- Quick fact lists and achievement summaries
- Each civilization's enduring mystery

## Local Deployment
```
cd "Lost Civilization Explorer"
start index.html
```
