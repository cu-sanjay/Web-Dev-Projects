# 🕵️ Fake Finder

## Description

Added Fake Finder, an educational fact-checking game where users are presented with four statements—three true and one false. The objective is to identify the fake fact using logic, knowledge, and critical thinking skills.

## Features

* Four-fact challenge system
* One fake fact among three real facts
* Multiple knowledge categories
* Score tracking
* Instant answer validation
* Progressive difficulty levels
* Educational fact-based gameplay

## Example

**Facts:**

1. The Earth revolves around the Sun. ✅
2. Water boils at 100°C at sea level. ✅
3. Humans have three hearts. ❌
4. Plants produce oxygen through photosynthesis. ✅

**Answer:** Humans have three hearts.
