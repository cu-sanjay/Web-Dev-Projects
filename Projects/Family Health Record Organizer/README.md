# 🏥 Family Health Record Organizer

A centralized digital dashboard for families to securely organize, manage, and access essential health information for multiple family members in one place.

## Features

- **Multi-Member Profiles**: Create detailed profiles for every family member (Spouse, Children, Parents) including their age, DOB, blood group, and allergy warnings.
- **Comprehensive Medical Records**: Track Medical History, Vaccinations, Current Medications, and Doctor Appointments individually for each family member.
- **Health Dashboard**: A high-level overview showing total family members, pending vaccinations, upcoming appointments, and a chronological timeline of recent health activity.
- **Emergency Contacts**: A dedicated, easily accessible section for primary healthcare providers, hospitals, and emergency phone numbers.
- **Premium Glassmorphism UI**: Beautiful, clean interface built with CSS variables, gradients, and frosted glass effects.
- **Customizable Themes**: Choose from 4 unique color themes (Ocean, Nature, Berry, Sunset) alongside a built-in Dark Mode toggle.
- **Data Persistence**: All records, profiles, and theme preferences are saved securely and locally in your browser's `LocalStorage`.

## How to Run

1. Clone the repository.
2. Navigate to the `Projects/family-health-record-organizer` folder.
3. Open `index.html` in your web browser.

## Technologies Used

- HTML5
- CSS3 (Custom Properties, CSS Grid, Flexbox, Glassmorphism)
- Vanilla JavaScript (LocalStorage API, DOM Manipulation)
