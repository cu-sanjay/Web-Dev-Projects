# 🌸 Pollinator Garden Planner

Drag-and-drop garden designer that suggests plants attracting bees and butterflies, with a seasonal bloom calendar.

## Features
- 🖱️ Drag-and-drop plant palette
- 🌻 Suggests pollinator-friendly plants
- 📅 Seasonal bloom calendar
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Drag plants from the palette into the garden area and view their bloom season in the calendar.
