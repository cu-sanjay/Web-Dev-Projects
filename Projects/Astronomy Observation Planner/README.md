# Astronomy Observation Planner

Interactive night sky planner displaying stars, constellations, planets, moon phase, and observing conditions based on latitude, month, and time.

## Features
- Dynamic star chart with 250+ stars
- Constellation lines with labels (Ursa Major, Orion, Cassiopeia, Scorpius)
- 4 visible planets with magnitudes
- Moon phase visualization
- Observing conditions indicator

## Local Deployment
```
cd "Astronomy Observation Planner"
start index.html
```
