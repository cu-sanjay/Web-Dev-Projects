#  Doodle Dash

## Description

Added Doodle Dash, an interactive drawing and guessing game where players either sketch a randomly assigned word or try to identify drawings created by others. The game combines creativity, quick thinking, and visual recognition in a fun and engaging experience.

## Features

* Random drawing prompts
* Drawing canvas system
* Guess-the-drawing gameplay
* Countdown timer
* Score tracking and calculation
* Multiple rounds and challenges
* Interactive drawing experience

## Benefits

* Encourages creativity and imagination
* Improves visual interpretation skills
* Enhances quick-thinking abilities
* Creates an engaging social gaming experience

## Future Improvements

* Multiplayer rooms
* Leaderboard system
* Achievement badges
* Custom word categories
* Drawing replay feature
* Daily drawing challenges

## Use Cases

* Educational activities
* Creative learning games
* Team-building exercises
* Fun social interactions

## Learning Outcome

Users improve creativity, communication, observation, and visual reasoning skills through drawing and guessing challenges.
