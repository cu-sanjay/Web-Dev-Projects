# Personal Finance Management Platform

A sleek and comprehensive front-end web application that helps users track personal expenses, manage monthly budgets, and analyze spending habits across different categories.

## Features
- **Financial Dashboard**: Get a high-level overview of total balance, monthly income, and monthly expenses.
- **Transaction Tracker**: Log and monitor all recent financial activities with an interactive table.
- **Budget Manager**: Keep track of budget limits across categories like Housing, Food, and Transportation with visual progress bars.
- **Goal Setting**: Set and monitor long-term financial goals (e.g., Emergency Fund, Vacation).
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
