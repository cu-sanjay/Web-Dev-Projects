# Skill Exchange Platform

A modern, interactive web platform that connects individuals looking to teach, learn, and trade skills. Build your reputation, offer your expertise, and request help from others within the community!

## Features
- **Dashboard**: Track your skill credits, view upcoming exchange sessions, and browse recent requests.
- **Skill Market**: A centralized feed where users can browse skills being offered and requested by the community.
- **My Offerings**: Manage the skills you are teaching and approve pending requests from other users.
- **Reputation System**: Earn skill credits by teaching others, which you can spend to learn new skills.
- **Responsive Design**: Fully responsive interface built with CSS Grid and Flexbox for mobile and desktop.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
