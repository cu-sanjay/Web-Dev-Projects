# Open Source Contribution Planner

Plan your open source journey with skill-based repository recommendations, a 30-day roadmap, and personalized configuration. Supports 7 programming languages and 3 experience levels.

## Features
- 7 programming language skills
- 3 experience levels (beginner, intermediate, advanced)
- 21 recommended repositories with labels
- 4-week progressive roadmap
- Hours-per-week budgeting

## Local Deployment
```
cd "Open Source Contribution Planner"
start index.html
```
