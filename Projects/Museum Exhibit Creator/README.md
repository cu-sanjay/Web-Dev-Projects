# Museum Exhibit Creator

Create virtual museum exhibits by selecting artifacts, arranging them in a 4-slot grid, choosing a theme, and publishing your exhibit.

## Features
- 6 artifact types with icons and descriptions
- 4-slot exhibit grid
- Theme selection (Egypt, Greece, Rome, China)
- Exhibit creation/publishing workflow

## Local Deployment
```
cd "Museum Exhibit Creator"
start index.html
```
