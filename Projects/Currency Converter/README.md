# 💱 Currency Converter

A real-time currency converter using a public exchange rate API with swap functionality and country flag display — built using HTML, CSS, and JavaScript.

## Features
- 🔄 Swap currencies instantly
- 🌍 Real-time exchange rates via API
- 🇺🇸 Country flag display for currencies
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Enter an amount, select currencies, and click **Convert**.
