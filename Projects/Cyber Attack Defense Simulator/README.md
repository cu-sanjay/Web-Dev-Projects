# Cyber Attack Defense Simulator

Network defense tower-defense style game. Deploy firewalls, antivirus, IDS, WAF, and backups to protect 5 network nodes from 8 attack types across progressive waves.

## Features
- 5 deployable defense systems
- 8 attack types (DDoS, Malware, SQL Injection, Phishing, Ransomware, Zero Day, Brute Force, XSS)
- 5 network nodes with individual health
- Wave progression with increasing difficulty
- Score and health tracking

## Local Deployment
```
cd "Cyber Attack Defense Simulator"
start index.html
```
