# Student Expense Sharing Manager

A practical web application designed to help friends and roommates track shared expenses, split bills evenly or unevenly, and effortlessly calculate who owes whom. 

## Features
- **Member Management**: Easily add and manage group members.
- **Expense Logging**: Record new expenses, select the payer, choose the participants involved, and specify the split amount.
- **Dynamic Balances**: The app automatically calculates running balances for every member, showing exactly who is in the green and who needs to pay up.
- **Transaction History**: View a chronological history of all group expenses.
- **Responsive Design**: Beautiful, mobile-friendly interface built with modern CSS techniques.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
