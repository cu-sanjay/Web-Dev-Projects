# Missing Person Search Simulator

Search simulator with a color-coded probability heatmap. Search 4 sectors (Urban, Forest, Coastal, Mountain) to discover clues and track search progress.

## Features
- 4 search sectors with different terrain types
- Dynamic probability heatmap (pixel-level rendering)
- Clue discovery system
- Search progress and probability tracking

## Local Deployment
```
cd "Missing Person Search Simulator"
start index.html
```
