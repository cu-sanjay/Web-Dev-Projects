# 🔢 Number Merge Quest


## Description

Number Merge Quest is an interactive puzzle game where players merge numbers strategically to reach a target value within limited moves.

The game improves logical thinking and problem-solving skills through progressively challenging number combinations.


## Features

- 🎮 Interactive gameplay
- 🔢 Number merging mechanics
- 🎯 Target-based challenges
- ⏳ Limited moves system
- 🔄 Restart option
- 💻 Built using Vanilla JavaScript


## How To Play

1. Click on any number tile.
2. The selected number merges with the next number.
3. The merged value increases.
4. Reach the target value before running out of moves.


## Technologies Used

- HTML5
- CSS3
- JavaScript
