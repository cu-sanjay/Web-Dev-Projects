# AI Study Planner & Learning Roadmap Builder

An interactive, premium client-side study planner that constructs custom week-by-week learning roadmaps, maps milestone stages using visual node pathways connected with SVG lines, and records study hours and notes persistently in browser storage.

## Features

- **Visual Roadmap Node Canvas**: Node pathways linked dynamically using responsive SVG connector lines representing learning progress.
- **Custom Study Pacer**: Configures study schedules based on topic domains (Frontend Dev, Machine Learning, DSA, DevOps, System Design), available weekly hours, and pacing styles.
- **Milestone Side Drawer**: Slide-in description drawer showing phase summaries, task lists, and note-taking scratchpads.
- **Performance Logs Tracker**: Analytics dashboard calculating overall roadmap completion percent, total study hours logged, and learning streaks.
- **Sleek Glassmorphism UI**: High-fidelity dark mode with smooth CSS panel transitions and interactive hover elements.

## Run it

Open `index.html` in any modern browser.

## What it shows

- Responsive SVG line connections connecting dynamic absolute-positioned elements.
- Scheduling math allocating hours and weeks dynamically based on course profiles.
- Integrated notepad saving state per node in local storage databases.
- UI sidebar sliders, visual progress gauges, and sliding drawer transitions.
- Semantic structure layout supporting keyboard navigation.
