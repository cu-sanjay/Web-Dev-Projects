# 🧮 Equation Fixer

## Description

Added Equation Fixer, an interactive mathematics puzzle game where users must move a single number or operator to make an incorrect equation correct. The game features an advanced color-changing UI that enhances engagement and provides a dynamic learning experience.

## Features

* Equation correction challenges
* Move one number or operator gameplay
* Advanced color-changing user interface
* Multiple difficulty levels
* Score tracking system
* Instant answer validation
* Progressive puzzle complexity

## Example

**Equation:** 5 + 3 = 10

**Move one operator:**
5 + 5 = 10 ✅

**Equation:** 8 - 3 = 10

**Move one number:**
8 + 2 = 10 ✅
