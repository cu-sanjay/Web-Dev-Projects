# OBSCURA

## Description

Added OBSCURA, a visual puzzle game where players identify and match objects with their corresponding shadows. The game tests observation, recognition, and visual reasoning skills through increasingly challenging object sets.

## Features

* Object-to-shadow matching gameplay
* Multiple object categories
* Score tracking system
* Hint system for difficult matches
* Progressive difficulty levels
* Instant answer validation
* Visual recognition challenges

## Benefits

* Improves observation skills
* Enhances visual perception and recognition
* Strengthens attention to detail
* Encourages logical and analytical thinking

## Future Improvements

* Timed challenge mode
* Achievement badges
* Leaderboard system
* Daily shadow puzzles
* Custom object packs
* Performance analytics dashboard

## Use Cases

* Brain-training activities
* Educational visual learning
* Cognitive skill development
* Interactive puzzle games

## Learning Outcome

Users develop stronger visual recognition, observation, and analytical thinking skills by identifying relationships between objects and their shadows.
