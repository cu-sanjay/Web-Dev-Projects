# 💊 Smart Medication Tracker & Pill Adherence Scheduler

An interactive client-side application designed to solve the real-world challenge of medication adherence. Track doses, design custom schedules, customize pill designs visually, receive alerts, and manage pill inventories.

---

## 🌟 Key Features

1. **Custom Dosing & Visual Pill Designer**:
   - Customize pill shapes (tablet, capsule, circular, liquid bottle) and color themes.
   - Design schedules based on daily, specific days of the week, or multiple custom times.

2. **Interactive Daily Timeline**:
   - Visual checklist sorted by time.
   - Quick check-off logs with smooth micro-animations.

3. **Smart Inventory & Refill Warnings**:
   - Track remaining quantities and receive alerts when stocks drop below a critical threshold.

4. **Analytics Dashboard**:
   - Visual weekly compliance logs.
   - Interactive adherence streaks and completion rate tracking.

5. **Audio Alarms & System Notifications**:
   - Built-in audio synth timers that trigger sound alerts when it's time to take your dose.

---

## 🚀 How to Run

1. Simply open [index.html](file:///index.html) in any modern web browser.
2. All data is saved automatically on your local machine using client-side `localStorage`.
