# College Life Simulator

A text-and-stat-based interactive simulator game where players step into the shoes of a college student trying to survive a semester. Make daily choices to balance your Academics, Social Life, and Health. Can you make it to finals week without dropping out?

## Features
- **Stat Tracking**: Real-time progress bars for Academics, Social, and Health stats.
- **Dynamic Events**: Encounter randomized daily events (like surprise quizzes, parties, or getting sick) that challenge your decision-making.
- **Consequence System**: Every choice affects your stats. Let one drop to 0, and it's game over!
- **Progression**: Survive through 15 "weeks" of college to win the game.
- **Responsive UI**: Clean, engaging interface playable on desktop and mobile.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
