# Research & Innovation Management Portal

A comprehensive front-end web application aimed at facilitating academic research and tracking scholarly innovation. The platform provides tools for managing research projects, tracking publications, and fostering collaboration among researchers.

## Features
- **Researcher Dashboard**: An overview of active projects, recent publications, and pending reviews.
- **Project Directory**: Browse ongoing research initiatives, clinical trials, and studies across different departments.
- **Publications Hub**: A dedicated interface for managing, reading, and submitting academic papers.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
