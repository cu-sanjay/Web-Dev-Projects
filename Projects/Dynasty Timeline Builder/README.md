# Dynasty Timeline Builder

Interactive timeline builder for 4 dynasties: Ming, Tudor, Ottoman, and Maurya. Each dynasty features color-coded events (birth, war, culture, death) with descriptions.

## Features
- 4 dynasties with comprehensive event timelines
- Color-coded event types
- Dynasty summary with dates and descriptions
- Scrollable timeline view

## Local Deployment
```
cd "Dynasty Timeline Builder"
start index.html
```
