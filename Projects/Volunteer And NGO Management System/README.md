# Volunteer And NGO Management System

A centralized platform designed to bridge the gap between passionate volunteers and impactful NGOs. It allows individuals to discover causes, track their volunteer hours, and apply for events, while giving NGOs the tools they need to manage campaigns and volunteers.

## Features
- **Discover Opportunities**: Browse upcoming events, campaigns, and local community initiatives.
- **Volunteer Dashboard**: Track your total impact hours, upcoming shifts, and badges earned.
- **NGO Dashboard**: Manage active campaigns, review volunteer applications, and coordinate event logistics.
- **Role Switching**: Toggle between the Volunteer view and the NGO Coordinator view to test both sides of the platform.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
