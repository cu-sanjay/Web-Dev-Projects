# B2B Wholesale Marketplace

A professional B2B e-commerce interface connecting wholesale suppliers with retail buyers for bulk purchasing and inventory management.

## Features
- **Supplier Dashboard**: View bulk orders, manage wholesale inventory catalogs, and track monthly sales volume.
- **Buyer View**: Browse supplier catalogs, negotiate bulk pricing, and add items to a wholesale cart.
- **Interactive UI**: Switch seamlessly between the Supplier Management dashboard and the Buyer Shopping experience using the role-switcher dropdown.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
