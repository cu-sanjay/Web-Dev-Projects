#  Room Recall

## Description

Added Room Recall, a visual memory game where users are shown a room or image for a limited time and must answer questions about details they observed. The game challenges memory, attention to detail, and observation skills.

## Features

* Timed image display system
* Observation-based challenges
* Detail recall questions
* Score tracking
* Progressive difficulty levels
* Instant answer validation
* Visual memory training

## Example

**Image displayed for 10 seconds**

Questions:

* What color was the chair?
* How many books were on the shelf?
* What object was on the table?

## Benefits

* Improves memory retention
* Enhances observation skills
* Strengthens attention to detail
* Encourages quick information processing
