# Personal CRM Dashboard

A lightweight, robust web application to help individuals keep track of their networking contacts, maintain interaction histories, and schedule follow-ups without the complexity of an enterprise CRM system.

## Features
- **Contact Management**: Easily add new contacts with names, companies, roles, and email addresses.
- **Interaction Logging**: Keep a timestamped log of your interactions (e.g., meetings, emails, phone calls) with specific contacts.
- **Follow-up Reminders**: Tag interactions that require follow-ups to stay on top of your professional network.
- **Dynamic Filtering**: Instantly search your contacts list by name or company to quickly find who you're looking for.
- **Responsive Layout**: Designed with modern CSS techniques to look great on desktop and mobile.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
