# World Language Distribution Map

An interactive browser-based visualization of the world's major languages, their speaker populations, geographic distribution, and key statistics.

## Features
- 12 major world languages with detailed profiles
- Interactive region map highlighting language distribution
- Search by language name or country
- Speaker counts, script types, writing systems
- Country-level distribution data

## Local Deployment
```
git clone <repository-url>
cd "World Language Distribution Map"
start index.html
```
