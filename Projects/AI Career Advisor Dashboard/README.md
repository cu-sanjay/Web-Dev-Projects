# AI Career Advisor Dashboard

An interactive, premium client-side career advisor hub that evaluates professional matching scores based on skills, interests, and resumes, visualizes growth trends and salary metrics, and organizes milestone checklists.

## Features

- **Algorithmic Career Matcher**: Compares skill scores and resume texts against pre-set career profiles to compute match accuracy percentages.
- **Industry Growth Telemetry**: Visualizes annual job growth, salary distributions, and market demands for tech roles.
- **Milestone Roadmap checklists**: Interactive path checklists to check off required certifications, with values persisted in `localStorage` per role.
- **Sleek Dark glassmorphic Dashboard**: Centered on sky teal and neon hot pink accent indicators, radial progress SVG gauges, and responsive grids.

## Run it

Open `index.html` in any modern browser.

## What it shows

- Implementing similarity matching calculations in JavaScript.
- Dynamic data renderings updating salary ranges and checklists.
- Local storage maps persisting checked roadmaps.
- High contrast CSS colors and screen-readable forms structures.
