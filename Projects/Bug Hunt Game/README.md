# 🐞 Bug Hunt Game

## 📌 Project Description
Bug Hunt Game is a mini interactive coding puzzle game where players identify and fix bugs in small code snippets by choosing the correct option.

---

## 🎮 How to Play
1. Open `index.html` in your browser (double-click it).
2. Read the question and observe the code snippet.
3. Select the option that correctly identifies the bug (or says "No bug").
4. Get instant feedback.
5. Click "Next" to move to the next challenge.

---

## 🛠 Features
- Interactive coding quiz format
- Multiple bug-based questions
- Instant feedback system
- Beginner-friendly JavaScript logic
