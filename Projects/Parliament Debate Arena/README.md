# Parliament Debate Arena

Step into the parliamentary chamber where you select debate topics, hear arguments from both sides, and cast votes to pass or reject motions.

## Features
- **Debate Topics** — 8 distinct policy topics with government and opposition arguments
- **Voting System** — Cast votes For, Against, or Abstain with weighted seat counts
- **Session Tracking** — Track government/opposition seats, session number, and motions passed

Open `index.html` in any browser to begin debating.
