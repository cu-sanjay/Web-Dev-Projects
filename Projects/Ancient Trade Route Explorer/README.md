# Ancient Trade Route Explorer

Interactive SVG-based map explorer for 4 ancient trade routes: Silk Road, Spice Route, Incense Route, and Amber Road. Each route shows the path, major cities, distance, era, and traded goods.

## Features
- 4 ancient trade routes with animated paths
- City markers with labels on interactive SVG map
- Route details (distance, era, traded goods)

## Local Deployment
```
cd "Ancient Trade Route Explorer"
start index.html
```
