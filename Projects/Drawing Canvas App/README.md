# 🎨 Drawing Canvas App

A browser-based drawing app with brush size, color picker, eraser, clear canvas, and PNG download options — built using HTML5 Canvas, CSS, and JavaScript.

## Features
- ✏️ Adjustable brush size
- 🎨 Color picker
- 🧽 Eraser tool
- 🗑️ Clear canvas
- 💾 Download drawing as PNG
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Draw on the canvas, change brush size/color, erase, clear, or download your artwork.
