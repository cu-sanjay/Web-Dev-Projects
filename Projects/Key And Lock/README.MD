#  Key & Lock

## Description

Added Key & Lock, a puzzle-adventure game where players solve challenges to collect keys and unlock new doors, rooms, and levels. As players progress, puzzles become more difficult, creating an engaging journey of exploration and problem-solving.

## Features

* Key collection system
* Door and level unlocking mechanics
* Multiple puzzle types
* Progressive level progression
* Increasing difficulty levels
* Score tracking
* Achievement-based advancement

## Gameplay Flow

1. Solve puzzles and challenges
2. Earn keys as rewards
3. Unlock doors and new levels
4. Face increasingly difficult puzzles
5. Reach the final stage and complete the adventure

## Benefits

* Enhances logical reasoning
* Improves problem-solving skills
* Encourages strategic thinking
* Provides a rewarding progression system
