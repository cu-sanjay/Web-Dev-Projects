# Word Counter

A clean, real‑time word and character counter. Type or paste any text, and instantly see word count, total characters (with spaces), and characters without spaces. Perfect for writers, students, and social media managers.

## Features

- **Real‑time counting** – updates as you type, no button presses.
- **Three metrics** – words, characters with spaces, characters without spaces.
- **Clear text** – one‑click reset.
- **Copy to clipboard** – quickly export your text.
- **Fully responsive** – works on desktop, tablet, and mobile.
- **No gradients, pure CSS** – human‑made interface with smooth interactions.

## Tech Used

- **HTML5** – semantic markup, accessible.
- **CSS3** – flexbox, custom border radius, no gradients, soft shadows.
- **JavaScript (ES6)** – event listeners, dynamic DOM updates, clipboard API.
