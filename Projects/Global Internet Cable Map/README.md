# Global Internet Cable Map

An interactive map of the world's major undersea internet cables, showing cable routes, landing points, and infrastructure details.

## Features
- 10 major undersea cable systems visualized on an SVG world map
- Click cables to view length, capacity, status, and landing points
- Click landing points to see connected cables
- Search cables and landing points
- Interactive highlight on selection

## Local Deployment
```
git clone <repository-url>
cd "Global Internet Cable Map"
start index.html
```
