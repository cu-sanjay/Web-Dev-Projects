# 🏘️ Community Resource Directory

A responsive Community Resource Directory that serves as a centralized platform for organizing and accessing important local resources like hospitals, pharmacies, emergency contacts, NGOs, and more.

## Features

- **Resource Directory**: Browse, search, and filter community resources easily.
- **Categorization**: Resources are split into helpful categories (Hospitals & Clinics, Emergency Services, NGOs, etc.).
- **Search & Discovery**: Instantly find resources using the keyword search or dropdown category filter.
- **Emergency Quick Access**: Fast access to critical emergency numbers displayed prominently on the side.
- **Bookmark System**: Save important resources to a persistent bookmarks list for quick later access.
- **Add Resource**: User-friendly modal form to add new resources to the directory.
- **Theme Support**: Includes both Dark and Light themes that persist using LocalStorage.
- **Data Persistence**: Uses LocalStorage to save directory additions, deletions, bookmarks, and theme choices across sessions.

## How to Run

1. Clone the repository.
2. Navigate to `Projects/community-resource-directory/`.
3. Open `index.html` in your web browser.

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Grid, custom properties)
- Vanilla JavaScript (LocalStorage API, DOM Manipulation)
