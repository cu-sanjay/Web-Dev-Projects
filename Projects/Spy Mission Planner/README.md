# Spy Mission Planner

Plan and launch spy missions. Choose from 4 operations with different risk/reward profiles, equip up to 5 gadgets to improve success odds, and launch missions with outcome simulation.

## Features
- 4 unique missions with locations and briefings
- 5 gadgets with risk-reduction effects
- Dynamic risk and success percentage calculation
- Mission launch with randomized outcomes

## Local Deployment
```
cd "Spy Mission Planner"
start index.html
```
