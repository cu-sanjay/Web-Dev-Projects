# 🧠 Memorize

## Description

Added Memorize, a memory-training game where users observe highlighted locations on a grid and then recreate the pattern from memory. The game helps improve concentration, visual memory, and recall skills through increasingly challenging grid layouts.

## Features

* Grid-based memory challenges
* Pattern memorization system
* Location recreation gameplay
* Multiple difficulty levels
* Score tracking
* Instant answer validation
* Progressive challenge progression

## Example

1. Display highlighted grid cells for a few seconds.
2. Hide the pattern.
3. User selects the remembered locations.
4. Compare answers and award points.

## Benefits

* Improves visual memory
* Enhances concentration and focus
* Strengthens pattern recognition skills
* Encourages quick recall and observation
