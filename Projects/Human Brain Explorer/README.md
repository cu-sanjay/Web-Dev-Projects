# Human Brain Explorer

Interactive SVG-based brain anatomy explorer. Click on different brain regions (frontal, parietal, temporal, occipital, cerebellum) to view detailed function descriptions and Brodmann area data.

## Features
- SVG brain diagram with 5 clickable regions
- Detailed function lists for each lobe
- Brodmann area references
- Responsive layout

## Local Deployment
```
cd "Human Brain Explorer"
start index.html
```
