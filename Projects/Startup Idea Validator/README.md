# Startup Idea Validator

Validate startup ideas with a multi-dimensional scoring system. Enter your idea name, description, industry, and target market to receive a score (0-100) across 6 dimensions with customized feedback.

## Features
- 6 validation dimensions with visual bars
- Customized feedback based on industry and market
- Animated score ring
- Seeded random for consistent results

## Local Deployment
```
cd "Startup Idea Validator"
start index.html
```
