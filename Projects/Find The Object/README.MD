#  Find The Object

## Description

Added Find The Object, a visual memory and observation game where users are shown a collection of objects for a short period. One object is then removed, and users must identify the missing item from memory.

## Features

* Object display system
* Random object removal
* Missing object identification challenge
* Score tracking
* Progressive difficulty levels
* Instant answer validation
* Visual memory training

## Example

**Objects Displayed:**
Apple, Book, Clock, Pen, Cup, Key, Ball, Phone, Bag, Bottle

**After Hiding One Object:**
Apple, Book, Clock, Pen, Cup, Key, Ball, Phone, Bottle

**Answer:** Bag
