# Digital Detective Notebook

Interactive detective notebook with four tabs: clue log, suspect profiles, relationship graph, and case progress tracking.

## Features
- 4-tab interface (Clues, Suspects, Graph, Progress)
- Add custom clues and suspect profiles
- Canvas-based relationship graph
- Case completion percentage tracking

## Local Deployment
```
cd "Digital Detective Notebook"
start index.html
```
