# Travel Planning And Itinerary Platform

A dynamic frontend web application designed to help users plan trips, create day-by-day itineraries, and manage their travel budget all in one place.

## Features
- **Trip Dashboard**: View your upcoming trips with beautiful cover images and quick stats.
- **Itinerary Builder**: Manage a day-by-day schedule with drag-and-drop style UI elements (mocked) for flights, hotels, and activities.
- **Budget Tracker**: Monitor your trip expenses and visually see how much budget you have remaining.
- **Interactive UI**: Navigate seamlessly between the My Trips overview, the detailed Itinerary view, and the Budget tab.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
