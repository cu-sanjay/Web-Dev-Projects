# Ancient Civilization Explorer

An interactive browser-based exploration of six major ancient civilizations: Egypt, Indus Valley, Mesopotamia, Greece, Rome, and China.

## Features
- 6 civilizations with detailed profiles
- Interactive timeline with key historical events
- Key statistics (duration, population, area, rulers)
- Major achievements per civilization
- Quick-select sidebar navigation

## Local Deployment
```
git clone <repository-url>
cd "Ancient Civilization Explorer"
start index.html
```
