# Emoji Match Mania

A fun memory matching game where players flip cards to find matching emoji pairs.

## Features

* Emoji-based memory challenge
* Move counter
* Timer tracking
* Restart game option
* Responsive and lightweight design

## How to Play

1. Click a card to reveal an emoji.
2. Click another card to find its matching pair.
3. Match all pairs to complete the game.
4. Try to finish with the fewest moves and fastest time.

## Technologies Used

* HTML
* CSS
* JavaScript
