# Hacker Escape Room

Terminal-themed escape room with 5 cybersecurity challenges: port scanning, base64 decryption, SQL injection, hex reversal, and privilege escalation. Complete all challenges before the 15-minute timer runs out.

## Features
- 5 progressive hacker challenges
- Interactive terminal with command input
- Port scan visualizer
- 15-minute countdown timer
- Hint system

## Local Deployment
```
cd "Hacker Escape Room"
start index.html
```
