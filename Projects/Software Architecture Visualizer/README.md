# Software Architecture Visualizer

Interactive canvas-based visualizer for 6 software architecture patterns: Microservices, Monolith, Layered, Event-Driven, Hexagonal, and CQRS. Each pattern includes a node-edge diagram, complexity/scalability ratings, and a color legend.

## Features
- 6 architecture patterns with auto-generated diagrams
- Color-coded node types (service, database, queue, gateway, cache)
- Complexity, scalability, and team fit ratings
- Toggleable legend

## Local Deployment
```
cd "Software Architecture Visualizer"
start index.html
```
