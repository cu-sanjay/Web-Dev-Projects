# 🔐 Random Password Generator

A secure random password generator with options for length, uppercase, lowercase, numbers, and symbols — with a one-click copy button.

## Features
- 🔢 Choose password length (4–32)
- 🔠 Include uppercase letters
- 🔡 Include lowercase letters
- 🔢 Include numbers
- 🔣 Include symbols
- 📋 One-click copy to clipboard
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Select options, generate a password, and copy it instantly.
