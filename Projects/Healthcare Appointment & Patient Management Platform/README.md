# Healthcare Appointment & Patient Management Platform

A comprehensive front-end web application aimed at streamlining clinic operations. The platform provides tools for booking appointments, managing patient records, and tracking daily clinic schedules.

## Features
- **Appointment Booking**: Intuitive interface for patients to schedule visits with available doctors.
- **Patient Dashboard**: Overview of upcoming appointments, medical history, and prescribed medications.
- **Admin/Doctor Dashboard**: A specialized view for medical staff to manage their daily schedule and access patient records.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
