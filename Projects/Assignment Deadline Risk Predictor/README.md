# Assignment Deadline Risk Predictor

A productivity tool designed for students to evaluate the risk level of missing assignment deadlines. By inputting the task complexity, estimated duration, and the number of days left until the deadline, the application calculates a risk score and provides actionable recommendations.

## Features
- **Risk Assessment Algorithm**: Calculates risk based on a combination of deadline proximity, task complexity, and estimated hours required.
- **Visual Indicators**: Displays risk levels using color-coded indicators (Low, Moderate, High, Critical).
- **Actionable Advice**: Provides tailored suggestions based on the calculated risk to help students manage their time effectively.
- **Assignment Tracking**: Keeps a running list of assessed assignments so you can prioritize at a glance.
- **Responsive UI**: Clean, accessible design that works flawlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
