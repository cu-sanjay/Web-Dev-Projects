# Smart Public Transport Planner

A dynamic, front-end web application that simulates a smart city's public transit routing system. Users can search for optimized routes between stations, compare different transit modes (Bus, Train, Subway), and estimate travel times and ticket costs.

## Features
- **Route Search**: Input an origin and destination to generate possible transit routes.
- **Multi-Modal Options**: See options across different modes of transportation, each with varying durations and costs.
- **Dynamic Cost Estimation**: Automatically calculates ticket prices based on the number of stops and transit type.
- **Interactive Map Placeholder**: A clean visual area designed to integrate with future mapping APIs (like Google Maps or Mapbox).
- **Responsive Interface**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
