# Split Bill Calculator

A clean, modern, and highly functional bill splitting tool built with vanilla HTML, CSS, and JavaScript.

## Features

- **Total Bill Input**: Enter the base amount of your bill.
- **Tip Selection**: Choose from 10%, 15%, 20% or enter a custom percentage.
- **Equal Splitting**: Automatically divide the total (bill + tip) by the number of people.
- **Unequal Splitting**: Manually assign specific amounts to each person with real-time validation to ensure the total is covered.
- **Live Updates**: All calculations happen instantly as you type.
- **Reset Option**: Clear all inputs and start over with a single click.
- **Responsive Design**: Works perfectly on mobile and desktop.

## How to use

1. Open `index.html` in your browser.
2. Enter the bill amount.
3. Select or enter a tip percentage.
4. Set the number of people.
5. (Optional) Toggle "Unequal Split Mode" to manually allocate costs.
6. Check the results in the summary card!

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- JavaScript (ES6+)
