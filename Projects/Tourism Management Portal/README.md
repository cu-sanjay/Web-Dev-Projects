# Tourism Management Portal

A full-featured frontend web application for managing travel bookings, exploring popular tour packages, and tracking user itineraries.

## Features
- **Discover Destinations**: Browse beautiful, curated travel packages with images and pricing.
- **My Bookings**: Track your upcoming trips and past travel history.
- **Admin Dashboard**: Manage active tour packages, review booking requests, and track revenue metrics.
- **Role Switching**: Toggle between the Customer view and the Admin view to experience the entire portal.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
