# Binary Decoder

A simple binary decoder tool to convert binary numbers to readable words.

## Features

- Plain text entry box where the user enters a binary line.
- immediate decoding of the entered text on the push of a button.
- Simple clutter free environment.

## Run it

Open `index.html` in any modern browser.
