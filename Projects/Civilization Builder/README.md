# Civilization Builder

A turn-based city-building strategy game. Build houses, farms, mines, markets, barracks, and libraries on a 6×6 grid. Manage gold, food, and population to grow your civilization.

## Features
- 6 building types with unique effects
- Turn-based economy simulation
- Population growth and starvation mechanics
- 6×6 buildable grid

## Local Deployment
```
cd "Civilization Builder"
start index.html
```
