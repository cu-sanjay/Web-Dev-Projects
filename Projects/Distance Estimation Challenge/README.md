# 📏 Distance Estimation Challenge

A gamified **Distance Estimation Challenge** built with HTML, CSS, and JavaScript.  
Players estimate the distance between two objects on screen, and scores are awarded based on accuracy.

---

## ✨ Features
- Randomly generated scenarios with two objects
- User inputs estimated distance
- System calculates actual distance
- Accuracy percentage displayed
- Score accumulates based on precision
- Leaderboard with localStorage persistence
- Attractive UI with gradients, animations, and hover effects

---

## 🚀 How to Play
1. Click **New Scenario** to generate two random objects.
2. Estimate the distance between them and enter your guess.
3. Click **Submit Guess** to see actual distance, accuracy, and score.
4. Track your progress in the leaderboard.
5. Reset anytime to start fresh.
