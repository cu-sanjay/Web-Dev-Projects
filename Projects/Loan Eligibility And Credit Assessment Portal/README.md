# Loan Eligibility And Credit Assessment Portal

A financial dashboard that streamlines the loan application process by connecting loan applicants with credit assessment officers.

## Features
- **Applicant Dashboard**: A clean interface for users to check their loan eligibility, submit loan applications, and view their credit score simulator.
- **Loan Officer Portal**: A secure dashboard for bank officials to review pending applications, assess credit risk, and approve or reject loans.
- **Interactive Role Switcher**: Seamlessly toggle between the Applicant View and the Loan Officer View to experience both sides of the platform.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
