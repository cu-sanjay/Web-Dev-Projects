# Scholarship & Financial Aid Management System

A comprehensive front-end web application that streamlines the process of finding, applying for, and tracking scholarships and financial aid opportunities. 

## Features
- **Student Dashboard**: Track active applications, total awarded aid, and upcoming deadlines in one place.
- **Scholarship Directory**: Browse and filter available scholarships by category, amount, and eligibility.
- **Application Tracker**: Monitor the status of submitted applications through a clear timeline interface.
- **Responsive UI**: Built with modern CSS to work seamlessly on desktop and mobile devices.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
