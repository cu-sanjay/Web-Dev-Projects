# 📊 Deep Work Productivity Dashboard

Track focus sessions, deep work hours, distractions, and productivity performance metrics.

## Features
- ▶️ Start and end focus sessions
- ⏱️ Track deep work hours
- 🏆 Count completed sessions
- ⚠️ Log distractions
- 📊 Visualize metrics with charts
- 💾 LocalStorage persistence

## Usage
Open `index.html` in your browser.  
Start sessions, log distractions, and view your productivity metrics.
