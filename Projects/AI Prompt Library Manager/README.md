# AI Prompt Library Manager

An interactive, premium client-side prompt catalog and sandbox playground that manages customized AI prompt templates, parses bracketed variables (like `[topic]` or `[format]`) into dynamic form controls, and outputs compiled copyable text blocks.

## Features

- **Dynamic Variable Interpolator**: Automatically parses any bracketed tokens inside prompt templates and displays custom text fields to compile final outputs.
- **Structured Prompts Catalog**: Organizes templates by categories (Coding, Writing, Marketing, Data Science, etc.) with custom target model suggestions (GPT-4, Claude, Gemini).
- **Interactive Sandbox Playground**: Load templates into the sandbox, edit configurations, track copy statistics, and test-compile formatting.
- **Portability Backups**: Supports importing and exporting prompt collections as JSON files.
- **Sleek Glassmorphic Dark UI**: Vibrant neon indigo accents, slide-out templates, and copy indicators.

## Run it

Open `index.html` in any modern browser.

## What it shows

- Using Regular Expression parsers to extract custom variables from text bodies.
- Dynamic DOM input form creation and live input event mappings.
- JSON conversions supporting local backup file transfers.
- Local storage lists mapping customized prompt configurations.
- Screen-reader accessible forms and high contrast ratios.
