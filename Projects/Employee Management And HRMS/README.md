# Employee Management And HRMS

A futuristic, glassmorphism-inspired HR portal to manage employee records, track attendance, and process leave requests.

## Features
- **Dashboard**: High-level metrics on total employees, on-leave status, and recent activity.
- **Employee Directory**: A beautifully styled grid showing all active employees, their roles, and contact info.
- **Leave Requests**: A dedicated table interface for HR managers to approve or reject pending time-off requests.
- **Premium UI/UX**: Designed with a sleek dark-glassmorphism aesthetic, featuring smooth gradients and blurred backdrops.
- **Responsive Layout**: Adapts perfectly to both desktop and mobile screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid, Flexbox, Backdrop-filter)
- Vanilla JavaScript
