# Shark Tank Simulator

Step into the tank as an investor. Evaluate startup pitches, negotiate equity deals, and build your investment portfolio.

## Features
- **Startup Pitches** — 8 unique startups with detailed financials and founder stories
- **Investor Decisions** — Make offers with custom amounts and equity, or pass
- **Portfolio Tracking** — Track cash, portfolio value, deals made, and episodes

Open `index.html` in any browser to start investing.
