#  Escape Game

## Description

Added Exam Hall Escape, an interactive puzzle adventure where players are trapped inside an exam hall and must solve a series of challenges to find clues and escape before time runs out.

## Features

* Math puzzle challenges
* Logic and reasoning riddles
* Hidden clue discovery system
* Countdown timer mode
* Multi-stage puzzle progression
* Score tracking
* Interactive escape-room gameplay

## Gameplay Flow

1. Solve math puzzles
2. Crack logic riddles
3. Discover hidden clues
4. Unlock new areas
5. Escape before the timer expires

## Benefits

* Enhances problem-solving skills
* Improves logical reasoning
* Strengthens mathematical thinking
* Encourages critical thinking under pressure
