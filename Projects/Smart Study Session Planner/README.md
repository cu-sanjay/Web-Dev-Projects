# Smart Study Session Planner

A productivity and education tool designed to help students optimize their study time. Instead of cramming, this application takes a list of topics and the total available study time, and automatically generates an optimized study schedule using the Pomodoro technique (25 minutes of focus, 5 minutes of rest).

## Features
- **Topic Input**: Add multiple subjects or topics you need to study.
- **Smart Time Distribution**: Input your total available study time (in hours/minutes) and the algorithm will distribute it equitably across your topics.
- **Pomodoro Breakdown**: The planner automatically breaks the distributed time into optimal 25-minute study blocks and 5-minute break blocks.
- **Interactive Timeline**: Visualizes your entire study session step-by-step so you know exactly what to focus on and when to take a break.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
