# Community Infrastructure Management Portal

A centralized dashboard allowing residents and city officials to collaboratively track, report, and resolve local infrastructure issues like potholes, broken streetlights, and damaged sidewalks.

## Features
- **Public Feed**: A public view where residents can see recently reported issues in their neighborhood and upvote them to raise priority.
- **Report Issue**: A clean form interface for residents to submit new infrastructure problems.
- **City Admin Dashboard**: A secure portal for city workers to manage incoming reports, dispatch repair crews, and update ticket statuses.
- **Interactive Role Switcher**: Seamlessly toggle between the Resident View and the City Admin View to experience both sides of the application.
- **Responsive Design**: Built with pure CSS for a clean, accessible layout on mobile and desktop screens.

## Run It
No backend or build process required. Simply open `index.html` in your web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid & Flexbox)
- Vanilla JavaScript
